# 필라테스 콘텐츠 공장 서비스

서비스 번호: 367
원본 ID: svc-070
rank / tier / score: 367 / B / 80
카테고리 / 니치: content / 필라테스
권장 채널: web-marketplace
플랫폼: Gumroad
결제 시스템: Gumroad
가격 앵커: ₩250,000 package + ₩400,000/mo

## 오퍼
Ghost 또는 로컬 도구를 활용해 필라테스의 콘텐츠 공장 흐름을 세팅하고 운영 문서까지 납품한다.

## 고객 문제
반복 CS 답변을 매번 새로 작성한다

## 첫 실행 기준
필라테스의 현재 업무 화면 1개와 샘플 데이터 10개를 받아 첫 MVP 범위를 고정한다.

## 납품물
- 콘텐츠 캘린더
- 업종별 검수표
- 금지 표현 목록
- 게시 전 승인 플로우
- 샘플 게시글 3종
- 성과 점검표

## 작업 단계
- 1. 게시 채널, 타깃 고객, 금지 표현, 승인자를 확정한다.
- 2. 업종별 검수표와 콘텐츠 구조를 먼저 만든다.
- 3. 샘플 게시글을 작성하고 고객 승인 흐름을 테스트한다.
- 4. 게시 후 수정, 보관, 재사용 규칙을 인수인계한다.

## 필수 보강
금지 표현 목록, 업종별 검수표, 게시 전 승인 플로우를 추가한다.

## 약점과 방어
업종 규제와 과장 표현 검수가 없으면 게시 후 리스크가 생긴다.

## 결제/채널 주의
- Gumroad: 초저가 ZIP, 문서, 템플릿. 주의: 수수료/정산 조건 확인.
- Payhip: 디지털 다운로드 스토어. 주의: 게이트웨이 연결 필요.
- Ko-fi Shop: 후원형 디지털 상품과 커미션. 주의: PayPal/Stripe 연결 국가 확인.
- 크몽 결제: 한국어 설치 대행/서비스 패키지. 주의: 플랫폼 정책과 수수료 확인.
- marketplace: 상품 페이지 → 샘플 이미지 → 고정가 패키지 → 후기 수집 → 자체 웹 전환. 주의: 플랫폼 수수료, 금지 서비스, 정산 주기, 환불 정책을 확인한다.

## 환불 및 성과 안내
실제 결제는 사용자 소유 계정에서만 진행합니다. 납품 범위, 환불 조건, 고객 자료 제공 지연 시 일정 변경 조건을 사전에 안내합니다. 이 서비스는 세팅과 운영 문서를 납품하지만 매출, 문의, 수익을 보장하지 않습니다.
