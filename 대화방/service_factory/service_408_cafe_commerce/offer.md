rank / tier / score: 408 / B / 78

가격 앵커: ₩1,500,000 setup + ₩500,000/mo

## 오퍼
Medusa 또는 로컬 도구를 활용해 카페의 커머스 백오피스 흐름을 세팅하고 운영 문서까지 납품한다.

## 고객 문제
광고비를 쓰지만 어떤 링크가 매출로 이어지는지 모른다

## 첫 실행 기준
카페의 현재 업무 화면 1개와 샘플 데이터 10개를 받아 첫 MVP 범위를 고정한다.

## 납품물
- 상품 등록표
- 주문 상태 정의서
- 재고/옵션 점검표
- 배송/환불 안내문
- 결제 링크 연결표
- 운영 인수인계 체크리스트

## 작업 단계
- 1. 상품, 옵션, 가격, 배송/환불 조건을 먼저 고정한다.
- 2. 주문 상태와 고객 알림 메시지를 정리한다.
- 3. 결제 링크와 상품 페이지의 연결 상태를 확인한다.
- 4. 품절, 옵션 변경, 환불 요청 대응 절차를 문서화한다.

## 필수 보강
결제/배송/환불/세금 범위를 별도 옵션으로 분리한다.

## 약점과 방어
결제, 배송, 환불, 세금 범위를 분리하지 않으면 스코프가 폭발한다.

## 결제/채널 주의

- internet-service: 로그인 없는 MVP → 고객별 워크스페이스 → 구독/인보이스 → 운영 모니터링. 주의: 개인정보, 데이터 보관, 장애 대응 SLA를 계약에 포함한다.

## 환불 및 성과 안내
실제 결제는 사용자 소유 계정에서만 진행합니다. 납품 범위, 환불 조건, 고객 자료 제공 지연 시 일정 변경 조건을 사전에 안내합니다. 이 서비스는 세팅과 운영 문서를 납품하지만 매출, 문의, 수익을 보장하지 않습니다.
