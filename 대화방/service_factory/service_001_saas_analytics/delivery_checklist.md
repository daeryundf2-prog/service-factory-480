# SaaS 분석 대시보드 납품 체크리스트

## 온보딩
- [ ] SaaS 제품 URL
- [ ] 현재 사용하는 분석 도구
- [ ] 핵심 퍼널 단계
- [ ] 가입/활성화/결제/이탈 정의
- [ ] 샘플 데이터 10건

## 전환 이벤트 사전
- [ ] signup_started
- [ ] signup_completed
- [ ] activation_event
- [ ] trial_started
- [ ] checkout_started
- [ ] payment_completed
- [ ] subscription_cancelled
- [ ] churn_risk_signal

## 데이터 품질 점검표
- [ ] 이벤트 이름 중복 없음
- [ ] 사용자 ID 연결 가능
- [ ] 날짜/시간대 일관성 확인
- [ ] 테스트 데이터 제외
- [ ] 결제 데이터와 이벤트 데이터의 기준 통일

## 월간 리포트 템플릿
- [ ] 방문자
- [ ] 가입 전환율
- [ ] 활성화율
- [ ] 결제 전환율
- [ ] 이탈률
- [ ] 다음 달 실험 제안 3개

## 납품 전 QA
- [ ] 데모 대시보드 열림
- [ ] 결제 링크가 사용자 소유 링크로 교체됨
- [ ] 환불/정산 문구 있음
- [ ] 개인정보/데이터 접근 권한 확인
