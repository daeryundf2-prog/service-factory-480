# 세무사 설문/폼 서비스

rank / tier / score: 108 / B / 86
카테고리 / 니치: forms / 세무사
권장 채널: web-marketplace
플랫폼: Payhip
결제 시스템: PayPal
가격 앵커: ₩350,000 setup + ₩80,000/mo

## 오퍼
Formbricks 또는 로컬 도구를 활용해 세무사의 설문/폼 흐름을 세팅하고 운영 문서까지 납품한다.

## 고객 문제
문의가 여러 채널로 흩어져 누락된다

## 첫 실행 기준
세무사의 현재 업무 화면 1개와 샘플 데이터 10개를 받아 첫 MVP 범위를 고정한다.

## 납품물
- 접수 폼 문항표
- 수집 동의 문구
- 스팸 방지 규칙
- CSV 백업 규칙
- 담당자 알림 흐름도
- 운영 인수인계 체크리스트

## 작업 단계
- 1. 수집 항목, 필수/선택 문항, 개인정보 동의 문구를 확정한다.
- 2. 접수 완료 메시지와 담당자 알림 흐름을 만든다.
- 3. 스팸, 중복, 빈 값, CSV 내보내기 테스트를 진행한다.
- 4. 백업 주기와 삭제 요청 대응 절차를 인수인계한다.

## 필수 보강
수집 동의 문구, 스팸 방지, CSV 백업 규칙을 추가한다.

## 약점과 방어
스팸 방지, 수집 동의, 저장 위치가 없으면 실제 운영에 취약하다.

## 결제/채널 주의
- Gumroad: 초저가 ZIP, 문서, 템플릿. 주의: 수수료/정산 조건 확인.
- Payhip: 디지털 다운로드 스토어. 주의: 게이트웨이 연결 필요.
- Ko-fi Shop: 후원형 디지털 상품과 커미션. 주의: PayPal/Stripe 연결 국가 확인.
- 크몽 결제: 한국어 설치 대행/서비스 패키지. 주의: 플랫폼 정책과 수수료 확인.
- marketplace: 상품 페이지 → 샘플 이미지 → 고정가 패키지 → 후기 수집 → 자체 웹 전환. 주의: 플랫폼 수수료, 금지 서비스, 정산 주기, 환불 정책을 확인한다.

## 환불 및 성과 안내
실제 결제는 사용자 소유 계정에서만 진행합니다. 납품 범위, 환불 조건, 고객 자료 제공 지연 시 일정 변경 조건을 사전에 안내합니다. 이 서비스는 세팅과 운영 문서를 납품하지만 매출, 문의, 수익을 보장하지 않습니다.
