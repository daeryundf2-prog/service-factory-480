# 서비스팩토리 영업패키지

## 패키지 구조
- 초기 세팅: 설치, 기본 설정, 화면 1개, 데이터 샘플, 사용법 안내.
- 월 유지보수: 업데이트 확인, 백업 확인, 장애 알림, 월간 리포트.
- 업셀: 추가 자동화, 추가 대시보드, 추가 채널, 직원 교육, 데이터 이전.

## 클라이언트에게 보내는 첫 문장
안녕하세요. 지금 문의/예약/고객상담/보고서 중 하나라도 수작업으로 처리하고 계시면, 오픈소스 도구 기반으로 이번 주 안에 작은 자동화 시스템을 하나 세팅해드릴 수 있습니다.

## 30초 제안문
현재 업무를 크게 바꾸지 않고, 반복되는 부분 하나만 자동화하는 방식입니다. 예를 들어 문의가 오면 시트에 기록되고, 담당자에게 알림이 가고, 첫 응대 문구까지 바로 복사할 수 있게 만듭니다. 첫 범위는 작게 잡고, 작동하면 월 유지보수로 이어갑니다.

## 가격표 예시
| 패키지 | 포함 | 가격 |
|---|---|---:|
| 문의 자동화 | 폼, 알림, 시트 기록, 응대문구 | ₩300,000부터 |
| 상담 인박스 | Chatwoot/FreeScout 설치, 채널 1개, 라벨 | ₩600,000부터 |
| CRM 세팅 | Twenty/EspoCRM 파이프라인, 필드, CSV 이전 | ₩800,000부터 |
| 대시보드 | Metabase/Umami/Plausible 이벤트와 리포트 | ₩500,000부터 |
| 내부툴 | Appsmith/ToolJet/Budibase 화면 1개 | ₩900,000부터 |
| 월 유지보수 | 업데이트, 백업, 알림, 요청 1시간 | ₩80,000부터 |

## 업셀 문구
- "이번에는 문의 흐름만 잡고, 다음 달에는 광고 리포트까지 붙이면 매주 확인 시간이 줄어듭니다."
- "설치만 하면 끝나는 도구가 아니라, 실제로 쓰는 화면과 문구까지 같이 맞춰드리겠습니다."
- "처음부터 큰 시스템을 만들지 않고, 돈이 새는 지점 하나만 먼저 막겠습니다."

## 업종별 첫 제안
### 병원/상담/학원
예약 변경, 사전 질문, 노쇼 방지 메시지를 자동화합니다.

### 쇼핑몰
반품 문의, 배송 문의, 리뷰 요청, 매출 리포트를 묶습니다.

### 프리랜서/에이전시
견적서, 청구서, 리드 관리, 프로젝트 상태 공유를 자동화합니다.

### B2B 영업팀
리드 CSV를 CRM으로 옮기고, 다음 액션과 후속 연락을 보이게 만듭니다.

## 무료 진단 질문
1. 하루에 반복해서 복사하거나 입력하는 일이 무엇인가요?
2. 문의가 들어온 뒤 첫 응답까지 보통 몇 분 걸리나요?
3. 지금 놓친 문의나 예약이 매출로 얼마나 아깝나요?
4. 가장 먼저 줄이고 싶은 업무 시간은 무엇인가요?
5. 이미 쓰는 도구가 구글시트, 이메일, 카톡, 슬랙 중 무엇인가요?

## 계약 전 주의문
오픈소스 도구는 무료로 시작할 수 있지만, 설치, 서버, 보안 업데이트, 백업, 장애 대응에는 운영 비용이 듭니다. 이 제안은 매출을 약속하는 것이 아니라 반복 업무를 줄이는 세팅과 운영 지원을 판매하는 것입니다.
